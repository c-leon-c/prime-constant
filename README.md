# prime-constant
A mathematical constant derived from paired square roots of consecutive primes.

## Formula

C = 1 + (sqrt(2) + sqrt(3))/10 + (sqrt(5) + sqrt(7))/100 + (sqrt(11) + sqrt(13))/1000 + ...


## Computation
- **~300,000 decimal digits** computed on Xiaomi smartphone
- **10,000 terms** in ~361 seconds using gmpy2
